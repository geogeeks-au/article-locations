Article Locations
=================

https://github.com/geogeeks-au/article-locations

Get a CSV of Wikipedia articles and their locations,
based on the coordinates of the articles or those of the articles' values for the following properties:

    'place of birth' => 'P19',
    'place of death' => 'P20',
    'residence' => 'P551',
    'work location' => 'P937'
